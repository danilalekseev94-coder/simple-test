# Simple CRM System with Telegram Integration

## Description
This project is a simple Customer Relationship Management (CRM) system integrated with Telegram. It uses Flask for the backend and `python-telegram-bot` for communication with Telegram.

## Getting Started
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Telegram Bot API token to `app.py` (replace `YOUR_TELEGRAM_BOT_TOKEN`).
4. Run the application: `python app.py`
